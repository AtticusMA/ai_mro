## 🐛 故障排查

### 问题：端口已被占用
**解决方案**：
```bash
# 使用其他端口启动
npm run dev -- --port 5176
```

### 问题：依赖安装失败
**解决方案**：
```bash
# 清除缓存并重新安装
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### 问题：Mock 数据不工作
**解决方案**：
- 检查浏览器控制台是否有错误
- 确保 Mock 接口路径与 API 调用路径一致
- 检查 `src/main.js` 中是否正确导入了 Mock

### 问题：路由不工作
**解决方案**：
- 检查 `src/router/index.js` 是否正确导入
- 确保 `src/main.js` 中正确使用了 router
- 检查路由定义中的路径是否正确
